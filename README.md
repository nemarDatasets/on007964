Raw EEG data and representational dissimilarity matrices can be found here.

Analysis scripts and behavioural data can be found via the open science framework: https://doi.org/10.17605/OSF.IO/JY284

A preprint of the manuscript can be found on biorxiv: https://doi.org/10.1101/2025.05.29.656895

Experiment Details: We recorded 128 channel EEG data from 20 observers as they viewed object images appearing at a rate of 3.33 Hz (image duration = 100 ms; ISI = 200 ms). One participant (sub-02) was excluded due to a technical error in the EEG recording, resulting in a final sample of 19 participants.
